# Bity API

> `https://swap.mewapi.io/bity`

## Bity API Methods

#### createTransaction
Request:
```
{
  "jsonrpc": "2.0",
  "method": "createTransaction",
  "params": {
    "amount": 0.5,
    "pair": "ETHBTC",
    "mode": 1,
    "destAddress": "1DECAF2uSpFTP4L1fAHR8GCLrPqdwdLse9"
  },
  "id": 83
 }
```

